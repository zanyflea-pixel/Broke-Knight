# Broke Knight (v24)

Run locally:
- Start a simple web server in this folder (recommended; ES modules need it):
  - Python: `python -m http.server 8000`
- Open: `http://localhost:8000/`

Controls:
- Move: Arrow keys
- Fireball: A
- Sail / Dock: B (only near docks)
- Map: M
- Inventory: I
- Skills: K
- God menu: G
- Save: Ctrl+S
- Load: Ctrl+L
